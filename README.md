# 🕐 BacaJam — Kuis Membaca Waktu

BacaJam adalah aplikasi kuis interaktif berbasis web untuk membantu pengguna belajar membaca dan memahami waktu dalam berbagai format:

* ⏱ Digital → Verbal
* 🗣 Verbal → Digital
* 🕐 Jam Analog
* 🎲 Mode Campuran

Dirancang sederhana, cepat, dan fokus untuk pembelajaran dasar.

---

## ✨ Fitur

### 🎯 Mode Kuis

* 10 soal per sesi
* Sistem skor:

  * ✅ Benar
  * ❌ Salah
  * 🔥 Streak
* Progress bar
* Feedback instan setelah menjawab
* Tombol petunjuk (hint)

### 📖 Materi Pembelajaran

* Pembagian waktu (pagi, siang, sore, dll)
* Konversi format 24 jam ke 12 jam
* Cara membaca jam analog
* Cara penyebutan waktu (setengah, kurang, dll)

### ⚡ Mode Tantangan

* Batas waktu 30 detik
* Soal acak cepat
* Skor real-time

## 🎮 Cara Menggunakan

1. Pilih mode kuis:

   * Digital → Verbal
   * Verbal → Digital
   * Analog
   * Campuran

2. Jawab soal yang muncul

3. Gunakan fitur:

   * 💡 Petunjuk → melihat hint
   * 🔄 Reset → mengulang kuis

4. Lihat hasil di akhir sesi

---

## 🧠 Contoh

* `20.14` → *Pukul 8 malam lewat 14 menit*
* *Setengah 8 pagi* → `07.30`
* Jam analog → pilih waktu yang sesuai
